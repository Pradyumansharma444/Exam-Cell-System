# Technology Stack - ExamCell System

## Frontend

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Components**: Shadcn/UI (Radix UI)
- **Icons**: Lucide React
- **State Management**: React Hooks & Server Actions

## Backend & API

- **Environment**: Node.js
- **API Architecture**: Next.js API Routes (Serverless)
- **Authentication**: JWT-based session management
- **Data Validation**: Zod / Server-side logic

## Database & Storage

- **Database**: PostgreSQL
- **Hosting**: Supabase
- **Client**: Supabase JS SDK / Postgres Pooler

## Third-Party Integrations

- **WhatsApp**: Twilio API (WhatsApp Business)
- **Email**: Resend API
- **Payments**: Razorpay
- **Deployment**: Vercel (with Daily Cron Jobs for Automation)
